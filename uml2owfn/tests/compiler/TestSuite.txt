process --------------------------------------------------------
  BOMcncpt1##ProcessPinDisjointStopEnd
test-case
  A process with multiple disjoint input/output pinsets relating
  input and output at the process level. The process can finish
  either via a stop node or via an end node. Depending on the
  non-deterministic choice, the output pinset is overridden. 
  
  This process is not sound.
