Quick install

If you have obtained a binary, copy the file into a folder of your choice. You may want to add this folder to the system path. If you have obtained the source code, extract it into a folder of your choice, and follow the steps below.

  1. Configure
    ./configure
  2. Compile
    make
  3. Test (optional, but recommended)
    make check
  4. Install
    sudo make install
